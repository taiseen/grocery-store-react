# Grocery Store - React | [Live Link](https://grocery-store-react-site.netlify.app)

## Using Tools & Language
* React
* SASS
* Swiper Slider 
* CDN - [Font-awesome](https://cdnjs.com/libraries/font-awesome) ( v-6.0.0 ) - CSS (icons)


## Package Installed
+ yarn create react-app
+ yarn add sass
+ yarn add swiper


## Run + Deploy
+ yarn start 
+ yarn build
